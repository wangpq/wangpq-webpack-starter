# webpack-starter

这是一个基础的webpack入门示例

这个示例向我们展示了如何使用sass, es6, 或者第三方库(jQuery, moment .etc), 或者使用jquery插件让webpack项目跑起来。

怎样运行示例?
**下载依赖到本地**
we are testing library using npm(jquery, moment) and bower (lodash).
```bash
npm install && bower install
```

**run the dev server**
```bash
npm run start
```
then open your http://localhost:8080 see how the magic happens.


**部署**

```bash
npm run build
```
然后你会发现一个叫build的新文件夹形成
